# AI vs Human Performance Comparison Analysis – Power BI

## Project Overview

This project analyzes the performance differences between Artificial Intelligence systems and humans across various tasks. Using Power BI, the data is visualized through interactive dashboards to compare efficiency, accuracy, and performance patterns.

## Dataset

The dataset contains information about different tasks performed by AI systems and humans. It includes performance metrics such as accuracy, completion time, and efficiency.

## Tools Used

* Power BI
* Excel
* Data Visualization

## Objectives

* Compare performance between AI and humans
* Identify areas where AI performs better than humans
* Analyze patterns and trends using visual dashboards

## Features

* Interactive Power BI dashboard
* Performance comparison charts
* Data insights through visual analysis
